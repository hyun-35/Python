# Jupyter Notebook



### 단축키

- `m` : markdown 입력창 → code 입력창
- `y` : code 입력창 → markdown 입력창

- `a` : 현재 셀 위에 추가
- `b` : 현재 셀 아래 추가
- `dd` : 현재 셀 삭제
- `z` : 셀 제거 취소
- `c` : 현재 셀 복사
- `Shift + v` : 현재 셀 위에 붙여넣기
- `v` : 현재 셀 아래 붙여넣기
- `Shift + Enter` : 현재 셀 실행 + 다음 셀로 이동
- `Ctrl  + Enter` : 현재 셀 실행
